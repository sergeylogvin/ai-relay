# Security Policy

## Supported versions

AI Relay is in early development. Security fixes are applied to the latest development version.

## Reporting a vulnerability

Do not create a public GitHub issue for vulnerabilities.

Use GitHub private vulnerability reporting when enabled, or contact the repository owner privately.

Include:

- affected component;
- reproduction steps;
- expected and observed behavior;
- potential impact;
- suggested mitigation, if known.

## Security principles

- no telemetry by default;
- no session-cookie extraction;
- no automatic message submission;
- local storage by default;
- explicit review before cross-provider transfer;
- minimum required permissions.
