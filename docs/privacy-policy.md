# AI Relay Privacy Policy

AI Relay is designed as a local-first browser extension.

It may process conversation text, titles, URLs, provider metadata, tags,
collections, saved views, preferences, local analytics, and duplicate
information.

Library data is stored locally in the user's browser profile by default. The
local Library does not intentionally transmit stored records to an external AI
Relay server.

Users can export, import, and delete Library records, remove the extension, and
clear extension storage through browser settings.
