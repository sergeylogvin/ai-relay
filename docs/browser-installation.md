# Install AI Relay Browser Extension

## Requirements

Users need a Chromium-based browser: Google Chrome, Microsoft Edge, or Brave.
Node.js and Git are not required when installing from a prepared ZIP archive.

## Install

1. Download the browser release ZIP.
2. Extract it to a permanent folder.
3. Open `chrome://extensions`, `edge://extensions`, or `brave://extensions`.
4. Enable **Developer mode**.
5. Click **Load unpacked**.
6. Select the extracted extension folder.
7. Pin AI Relay to the browser toolbar.

Do not delete or move the extracted folder while the extension is installed.

## Upgrade

Replace the extracted files with the newer release, then click **Reload** on
the browser extensions page.

## Uninstall

Open the browser extensions page and click **Remove** for AI Relay.
