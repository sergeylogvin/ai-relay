# Library v1 release notes

Library v1 turns the local conversation archive into a complete organization,
search, maintenance, and analysis workspace.

## Highlights

### Organization

- Tags and pinned conversations
- Collections and collection management UI
- Saved views for reusable search and filter configurations
- Local settings and preferences

### Search and discovery

- Ranked smart search across metadata and content
- Advanced filtering by provider, domain, tag, collection, dates, state, and
  duplicate status
- Analytics and local insights

### Data management

- Duplicate detection and review
- Merge and cleanup planning
- Bulk operations
- Versioned snapshot export and validated import
- Import/export browser UI

### Productivity

- Keyboard shortcuts
- Searchable command palette
- Local-first browser storage
- Stable, versioned data structures

### Accessibility and UX

- Skip navigation
- Focus-visible styling
- Focus trapping and restoration
- Screen-reader announcements
- Empty states and loading skeletons
- Reduced-motion and forced-colors support

## Privacy

Library processing and persistence remain local by default. No automatic
deletion, merge, cleanup, import persistence, or remote synchronization is
performed without an explicit user action.
